## 🚀 Overview

**Furi-Dashboard** is built with speed, aesthetics, and user experience in mind. It provides an intuitive interface to monitor, manage, and interact with your data in real-time. Driven by a robust Node.js backend, Furi gives you full control with style.

### ✨ Key Features

* **📊 Dynamic Visuals:** Real-time updating charts and widgets.
* **🎨 Ultra-Modern UI:** Clean layout with beautiful dark/light mode support.
* **⚡ Node.js Powered:** Event-driven, fast, and highly scalable architecture.
* **📱 Fully Responsive:** Seamless experience across desktop, tablet, and mobile devices.

## ⚙️ Installation & Setup
Get your local development environment up and running in just a few steps.
### Prerequisites
Before starting, make sure you have the following installed:
* [Git](https://git-scm.com/)
* [Node.js](https://nodejs.org/) (v18 or higher recommended)
### Step-by-Step Guide
1. **Clone the repository**
   ```bash
      git clone [https://github.com/FrostX2/Furi-Dashboard.git](https://github.com/FrostX2/Furi-Dashboard.git)
      cd Furi-Dashboard
2. **Configure Environment Variables**
   ```bash
      cp .env.example .env
Edit `PASSWORD` and `API` to whatever you want
3. **Install and run**
   ```bash
      npm install && node src/index.js
